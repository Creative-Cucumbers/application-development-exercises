# AGENTS.md

## このリポジトリについて

アプリケーション開発演習の講義資料をまとめたリポジトリ。

## 読者

- **学生**：各 Day の readme.md や資料を直接参照する。資料の文体・説明レベルは学生向けに合わせる。
- **講師（自分）**：授業設計・スライド・演習テーマの管理に使う。

## 構成

```
2026/
  1_〜/   Day1 資料
  ...
  6_comprehensive_exercise_theme_design_and_implementation/
          Day6・Day7 総合演習（readme.md が学生向け説明を兼ねる）
tools/    補助ツール類
```

## 演習の前提知識

- 対象：アプリケーション開発を学ぶ学生（約20名）
- 使用技術：Web フロントエンド、バックエンド API、データベース、認証、外部 API、micro:bit
- micro:bit・データベース・認証・外部 API は必須ではなくオプション
- Day6：UI 実装とデプロイ、Day7：改善と全員発表

## 資料を編集するときの注意

- readme.md などは学生も読む。専門用語は説明を添えるか、平易な表現にする。
- 「〜してください」「〜します」など、学生に向けた口調を維持する。
